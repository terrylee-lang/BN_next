## 人名與公司名規範

**公司行號名稱**：
- 有官方中文名稱者，**一律以中文名稱為主**，英文原文以括號附註於第一次出現時
  - 例：大鉦資本 (Centurium Capital)、輝達 (NVIDIA)、雀巢 (Nestlé)
- 無官方中文名稱者：保留英文原文，不強行音譯或意譯

**人名**：
- **絕對禁止**依外媒英文拼音自行音譯中文人名，錯誤示範：Li Hui → 李輝（實為黎輝）
- 查證順序：① 當事人官方資料或公司公告 ② 台灣主流媒體已使用之譯名 ③ 中國官方或當事媒體
- 查不到可靠中文名稱時：**保留英文原名**，並在文末或編輯附註中標記「人名待確認」
- 有公認譯名者：第一次出現使用「中文譯名 (英文原文)」格式
  - 例：伊隆·馬斯克 (Elon Musk)

## 禁止出現的寫法

| 類型 | 例子 | 上限 |
|------|------|------|
| 空洞開場白 | 「值得注意的是」、「不可否認」、「毫無疑問」、「顯而易見」 | 全文 0 次 |
| AI 式總結句 | 「這一切都指向一個事實：」、「歸根究柢，問題在於……」 | 全文 0 次 |
| 行銷形容詞 | 「革命性」、「顛覆性」、「里程碑」、「劃時代」（無具體數據依據時） | 全文 0 次 |
| 翻譯腔 | 「於此同時」作為固定句型、「對此」作為段落起手式 | 全文 0 次 |
| 空指稱弱化詞 | 「可以說」、「某種程度上」、「在多數情況下」 | 全文 0 次 |
| 破折號濫用 | 用於堆疊形容詞或情緒渲染 | 每 600 字最多 2 次 |

## 署名格式

文末資料來源與署名固定格式：

```
---

資料來源：[來源名稱](URL)

本文初稿為AI編撰，整理．編輯/ XXX
```

多來源時以「、」分隔，名稱加超連結，不裸露 URL。
